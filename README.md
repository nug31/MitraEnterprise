# 🏢 Mitra Enterprise

Website corporate modern untuk Mitra Enterprise dengan desain yang memukau dan fitur interaktif menggunakan React + TypeScript.

## ✨ Fitur

- 🎨 **Desain Modern**: Interface dengan gradien animasi dan efek glass morphism
- 📱 **Responsive Design**: Tampilan optimal di semua perangkat
- 🌐 **Bahasa Indonesia**: Terjemahan lengkap untuk target market lokal
- ⚡ **Performance**: Built dengan Vite untuk loading yang cepat
- 🎯 **Interactive**: Hover effects, animasi, dan micro-interactions
- 🏪 **Multi-Business**: Showcasing 7 unit bisnis yang berbeda

## 🏢 Unit Bisnis

1. **Mitra Garage** - Layanan otomotif profesional
2. **Mitra Coffee** - Kedai kopi premium 
3. **Hidroponik** - Pertanian modern hidroponik
4. **Mitra Motor** - Penjualan dan service motor
5. **Mitra Store** - Retail satu atap
6. **TEFA Mesin** - Layanan mesin industri
7. **Sunny Gold** - Perdagangan emas dan perhiasan

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Ready untuk Vercel/Netlify

## 🚀 Quick Start

### Prerequisites
- Node.js 16+ 
- npm atau yarn

### Installation

```bash
# Clone repository
git clone https://github.com/nug31/MitraEnterprise.git
cd MitraEnterprise

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build untuk Production

```bash
# Build aplikasi
npm run build

# Preview build
npm run preview
```

```bash

```

```bash
npm run build
# Deploy dist/ folder to gh-pages branch
```
